# flappy-pokemon
* 語言: java
* 開發軟體: eclipse
* 內容: flappy bird結合射擊遊戲

# 負責部分
  * Battle mode
  * 遊戲射擊功能
  * 圖片
 
# 操作方式
* 一般模式

 * 滑鼠左鍵跳躍

空白件攻擊
撞到敵人、水管、離開畫面Game Over
* Battle模式
P1
Q攻擊
W跳躍
P2
O攻擊
P跳躍
撞到敵人攻擊、水管、離開畫面為輸
* 階梯模式
按左右鑑移動
撞到紅方塊、離開畫面Game Over
